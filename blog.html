Question no.1:
Discuss the scope of var, let, and const.

Var:

  • Var declaration are globally scoped or function scoped.
  • var variables can be updated and re-declare within its scope
  • Var can be declared without being initialized.

Let:

  • Variables declared with 'let' are block-scoped
  • Let variables can be updated but not declared.
  • Let can be redeclare but cannot reinitialized

Const:

  • Also const are block scoped
  • Const is a block scope that can be accessible within block
  • Const variables can neither but not re-declared and cannot be reinitialized.

Question no.2:
Tell us the use cases of null and undefined

Null:

  • It means an empty value or a blank value and it is also a primitive type in javaScript.
  • The variable which has been assigned as 'null' contains no value
  • Null can be defined in object so that it can understand that the object should be clear and reset.

Undefined:

  • Undefined means the variable has been declared but its value has not been assigned.
  • When we do not pass an argument for a function parameter,the default is taken as 'Undefined'
  • When a function does not return a value,it returns undefined'
  • When trying to access an object property that does not exist,JavaScrips returns 'undefined'

Question no.3:
What do you mean by REST API?

REST API:

  • REST API stand for Representational State Transfer Application Programming Interface
  • A REST API is an API that conforms to the design principles of the REST.
  • REST is often implemented using HTTP as the communication protocol.
  • It is popular and widely used approach for building web services that allow different software application to communicate and exchange data over the internet.
